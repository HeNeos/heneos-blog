Project Euler 134

#project-euler#algorithms#number-theory

This is a simple problem, even when it has 45%45\%45% of difficult rating, even the code is short. There are no many concepts involved, just modular operations.

Statement

Consider the consecutive primes p1=19p_1 = 19p1​=19 and p2=23p_2 = 23p2​=23. It can be verified that 121912191219 is the smallest number such that the last digits are formed by p1p_1p1​ whilst also being divisible by p2p_2p2​.

In fact, with the exception of p1=3p_1 = 3p1​=3 and p2=5p_2 = 5p2​=5, for every pair of consecutive primes p2>p1p_2 > p_1p2​>p1​, there exist values of nnn for which the last digits are formed by p1p_1p1​ and nnn is divisible by p2p_2p2​. Let SSS be the smallest of these values of nnn.

Find ∑S\sum S∑S for every pair of consecutive primes with 5≤p1≤10000005 \leq p_1 \leq 10000005≤p1​≤1000000.

Solution

Let's take a look to the number that ends with the digits of p1p_1p1​:

np1‾=n⋅10⌊log⁡10p1⌋+1+p1\overline{n p_{1}} = n\cdot 10^{\lfloor \log_{10} p_{1}\rfloor + 1} + p_{1}np1​​=n⋅10⌊log10​p1​⌋+1+p1​

by the condition of the problem, this number should be divisible by p2p_2p2​:

n⋅10⌊log⁡10p1⌋+1+p1≡0mod  p2n≡−p1⋅inv(10⌊log⁡10p1⌋+1)mod  p2n\cdot 10^{\lfloor \log_{10} p_{1}\rfloor + 1} + p_{1} \equiv 0 \mod p_{2}\\ n \equiv -p_{1}\cdot \mathrm{inv}\big(10^{\lfloor \log_{10} p_{1}\rfloor + 1}\big) \mod p_{2}n⋅10⌊log10​p1​⌋+1+p1​≡0modp2​n≡−p1​⋅inv(10⌊log10​p1​⌋+1)modp2​

since the problem requires to find the minimum number, nnn must be the smallest positive value that satisfies the above equation.

The time complexity of this solution is O(Nlog⁡log⁡N+π(N)log⁡(N))\mathcal{O}(N\log\log N + \pi(N)\log(N))O(NloglogN+π(N)log(N)), where the value of π(N)\pi(N)π(N) could be bounded by x/ln⁡(x)x/\ln(x)x/ln(x), so the time complexity is: O(Nlog⁡log⁡N+N)\mathcal{O}(N\log\log N + N)O(NloglogN+N)

Code

#include <bits/stdc++.h>
using namespace std;
using ll = long long;

#define N 1000205

bitset <N> primes;
vector <pair<int, int> > p;

ll fastexp(ll x, ll y, ll p){
    ll ans = 1;
    while(y > 0){
        if(y&1) ans = (ans*x)%p;
        y >>= 1;
        x = (x*x)%p;
    }
    return ans%p;
}

ll inv(ll x, ll p){
    return fastexp(x, p-2, p);
}

void sieve(){
    int prev = 10;
    for(int i=2; i<N; i++){
        if(!primes[i]){
            if(i > prev) prev *= 10;
            p.push_back({i, prev});
            for(ll j=1LL*i*i; j<N; j+=i) primes[j] = 1;
        }
    }
}


int main(){
    sieve();
    ll ans = 0;
    for(int i=2; i<p.size()-1; i++){
        if(p[i].first > 1000000) break;
        int p1 = p[i].first;
        int p2 = p[i+1].first;
        int n = (1LL*inv(p[i].second, p2)*(p2 - p1)) % p2;
        if(n <= 0) n += p2;
        ans += 1LL*n*p[i].second + p[i].first;
    }
    cout << ans << '\n';
    return 0;
}
